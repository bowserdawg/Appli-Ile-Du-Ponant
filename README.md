## Présentation de l'appli

Cette appli permet à une personne de créer une réservervation pour un aller et possiblement un retour vers les îles du ponant, 
pour une ou plusieurs personnes, des animaux de compagnies et même des types de véhicules, à une date spécifique en Novembre 2025 (mais peut aussi prendre d’autres dates grâce aux fichiers CSV).

L'appli est divisé en deux : La première partie est en mode console avec C#, elle permet de créer une réservation avec un JSON spécifique. 

Il y'a aussi une partie WEB, qui permet de consulter en ligne plusieurs informations, comme:
  - Les horaires.
  - Les destinations.
  - Les cartes de réservation.
  - Les nombres de places restantes, pour véhicule et passage.
  - Des statistiques.

## Tutoriel installation

### Tuto partie console
Il faut avoir accés à mcs (compilateur C#).
Il faut aller dans le repertoire CSharp et taper 'mcs main.cs'.
